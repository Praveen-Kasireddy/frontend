INTRODUCTION

Welcome to KPMG Cosmos. The platform has been designed as a modern web application with the following core structure:

  • The navigation menu is located on the left sidebar. In addition, the user ID and system language are displayed at the bottom of the left sidebar. The sidebar expands to display the name of the icons in KPMG Cosmos when you hover your mouse over the sidebar.
  • Being a web application, KPMG Cosmos sessions timeout after 15 minutes of inactivity. The remaining time of an active session is displayed in the left sidebar. It is therefore necessary to save your work regularly.
  • Guidance is provided in the right sidebar.

MY PROJECTS OVERVIEW

The dashboard displays all projects on which you are a team member. In addition to the client, project name and due date, the project status (based on the number of completed task) is presented in the overview.

Select Create new project to start with the project setup.

Select a card in the dashboard to enter an existing project.
